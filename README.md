## Project Setup

```sh
pnpm i
```

### Compile and Hot-Reload for Development

```sh
pnpm tauri dev
```

### Type-Check, Compile and Minify for Production

```sh
pnpm tauri build
```

### Lint with [ESLint](https://eslint.org/)

```sh
pnpm lint
```
### Format with [Prettier](https://prettier.io/)

```sh
pnpm format
```
